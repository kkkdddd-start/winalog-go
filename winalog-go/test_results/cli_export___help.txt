Export events to various formats.

Usage:
  winalog export [command]

Available Commands:
  csv         Export to CSV
  json        Export to JSON
  timeline    Export timeline

Flags:
      --format string   Export format: csv, json, excel (default "csv")
  -h, --help            help for export
      --limit int       Maximum number of events to export (default 10000)

Global Flags:
      --config string      Config file path
      --db string          Database path
      --log-level string   Log level (debug/info/warn/error)

Use "winalog export [command] --help" for more information about a command.
