Perform forensics operations.

Usage:
  winalog forensics [command]

Available Commands:
  collect     Collect forensics data
  hash        Calculate file hash
  verify      Verify file signature

Flags:
  -h, --help   help for forensics

Global Flags:
      --config string      Config file path
      --db string          Database path
      --log-level string   Log level (debug/info/warn/error)

Use "winalog forensics [command] --help" for more information about a command.
