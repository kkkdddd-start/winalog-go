Build and query global event timelines.

Usage:
  winalog timeline [command]

Available Commands:
  build       Build global timeline
  query       Query timeline

Flags:
      --category string   Filter by category
      --computer string   Filter by computer
      --end string        End time (RFC3339)
  -h, --help              help for timeline
      --start string      Start time (RFC3339)

Global Flags:
      --config string      Config file path
      --db string          Database path
      --log-level string   Log level (debug/info/warn/error)

Use "winalog timeline [command] --help" for more information about a command.
