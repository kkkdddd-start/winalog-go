Import Windows event log files into the database for analysis.

Supported file types:
  - .evtx (Windows Event Log)
  - .etl (Event Trace Log)
  - .log (Text/CSV logs)
  - .csv (Comma-separated values)
  - Sysmon format logs

Examples:
  winalog import security.evtx
  winalog import --log-name Security security.evtx system.evtx
  winalog import --incremental --workers 4 ./logs/

Usage:
  winalog import [flags] <path>

Flags:
      --alert-on-import        Trigger alert analysis after import
      --batch-size int         Batch size for insertion (default 10000)
  -h, --help                   help for import
      --incremental            Enable incremental import (default true)
      --log-name string        Log name for imported files
      --skip-patterns string   Patterns to skip (comma-separated)
      --workers int            Number of parallel workers (default 4)

Global Flags:
      --config string      Config file path
      --db string          Database path
      --log-level string   Log level (debug/info/warn/error)
