User behavior analytics to detect anomalous user activities such as impossible travel, abnormal behavior, and privilege escalation.

Usage:
  winalog ueba [command]

Available Commands:
  analyze     Run UEBA analysis
  baseline    Manage user baselines
  profiles    Show user behavior profiles

Flags:
      --action string   Baseline action: learn, clear, show (default "show")
  -h, --help            help for ueba
  -H, --hours int       Analysis time window in hours (default 24)
      --save-alerts     Save detected anomalies as alerts

Global Flags:
      --config string      Config file path
      --db string          Database path
      --log-level string   Log level (debug/info/warn/error)

Use "winalog ueba [command] --help" for more information about a command.
