Convert Windows Event Log (EVTX) files to CSV format without storing in database.

Examples:
  winalog evtx2csv security.evtx security.csv
  winalog evtx2csv -l 1000 security.evtx > security.csv

Usage:
  winalog evtx2csv <input.evtx> [output.csv] [flags]

Flags:
  -h, --help        help for evtx2csv
  -l, --limit int   Limit number of events (0 = no limit)

Global Flags:
      --config string      Config file path
      --db string          Database path
      --log-level string   Log level (debug/info/warn/error)
