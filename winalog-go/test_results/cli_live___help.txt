Real-time event log monitoring.

Usage:
  winalog live [command]

Available Commands:
  collect     Start live collection

Flags:
  -h, --help   help for live

Global Flags:
      --config string      Config file path
      --db string          Database path
      --log-level string   Log level (debug/info/warn/error)

Use "winalog live [command] --help" for more information about a command.
