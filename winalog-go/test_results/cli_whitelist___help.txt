Manage whitelist rules to suppress specific alerts.

Usage:
  winalog whitelist [command]

Available Commands:
  add         Add a whitelist rule
  list        List all whitelist rules
  remove      Remove a whitelist rule

Flags:
  -h, --help   help for whitelist

Global Flags:
      --config string      Config file path
      --db string          Database path
      --log-level string   Log level (debug/info/warn/error)

Use "winalog whitelist [command] --help" for more information about a command.
