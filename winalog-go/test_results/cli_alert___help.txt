Manage and view security alerts

Usage:
  winalog alert [command]

Available Commands:
  delete      Delete an alert
  export      Export alerts to JSON
  list        List alerts
  monitor     Run in continuous alert monitoring mode
  resolve     Mark alert as resolved
  run         Run alert analysis on stored events
  show        Show alert details
  stats       Show alert statistics

Flags:
  -h, --help   help for alert

Global Flags:
      --config string      Config file path
      --db string          Database path
      --log-level string   Log level (debug/info/warn/error)

Use "winalog alert [command] --help" for more information about a command.
