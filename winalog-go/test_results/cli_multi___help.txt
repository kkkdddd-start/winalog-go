Cross-machine correlation and lateral movement detection.

Usage:
  winalog multi [command]

Available Commands:
  analyze     Analyze cross-machine
  lateral     Detect lateral movement

Flags:
  -h, --help   help for multi

Global Flags:
      --config string      Config file path
      --db string          Database path
      --log-level string   Log level (debug/info/warn/error)

Use "winalog multi [command] --help" for more information about a command.
