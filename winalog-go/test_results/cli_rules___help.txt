Manage alert and correlation rules.

Usage:
  winalog rules [command]

Available Commands:
  disable     Disable a rule
  enable      Enable a rule
  list        List all rules
  status      Show rule status (or all rules if no name provided)
  validate    Validate rule file

Flags:
  -h, --help   help for rules

Global Flags:
      --config string      Config file path
      --db string          Database path
      --log-level string   Log level (debug/info/warn/error)

Use "winalog rules [command] --help" for more information about a command.
