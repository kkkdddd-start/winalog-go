Generate security and analysis reports.

Usage:
  winalog report [command]

Available Commands:
  generate    Generate a report

Flags:
      --format string       Report format: html, json (default "html")
  -h, --help                help for report
  -o, --output string       Output file
      --time-range string   Time range for report (default "24h")

Global Flags:
      --config string      Config file path
      --db string          Database path
      --log-level string   Log level (debug/info/warn/error)

Use "winalog report [command] --help" for more information about a command.
